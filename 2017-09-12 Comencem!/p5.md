## Operacions sobre els enters (`int`)

| Operador | Operació |
|----------|----------|
| `+`      | suma     |
| `-`      | resta     |
| `*`      | producte     |
| `/`      | divisió entera  |
| `%`      | rest de la divisió entera  |

## Comparació entre els enters

| Operador | Operació |
|----------|----------|
| `<`      | més petit     |
| `>`      | més gran     |
| `<=`      | més petit o igual |
| `>=`      | més gran o igual |
| `==`      | igual  |
| `!=`      | diferetnt |
